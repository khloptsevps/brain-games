# "Brain Games"

## Проект "Игры разума"

Учебный проект школы программирования ["Hexlet"](https://ru.hexlet.io/?ref=252944).  
Содержит пять простых, математических игр.

## Setup

```sh
$ make install
```

## start:

```sh
$ make start
```

## how to play: "Brain-even"

```sh 
$ make brain-even
```
